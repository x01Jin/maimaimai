How to Use maimaimai
Add Match

Click the blue + button to add players to the queue. Enter Player 1 name (required) and Player 2 name (optional for solo play).

Start Match

When it's a player's turn, click the green play button next to their name to start their match.

Edit Match

Click the blue edit button to modify player names or change between solo/VS mode.

Remove Match

Click the red trash button to remove a match from the queue.

Reorder Queue

Drag and drop matches using the grip handle to reorder the queue.

Complete Match

When the current match is finished, click "Complete" to move to the next player in queue.
